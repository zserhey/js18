Стили на всех h2 должны быть одинаковы
должны использоваться правильные (семантические) теги
валидация должна быть зеленой
горизонтального скрола не должно быть
меньше чем 320px страница ужиматься не должна
ховер на кнопках обязателен 


1) Меню <768px должно перестроится в мобильное меню и открываться по клику


2) Секция с формой:
	- на секции должен быть фон, кототрый заполняет все пространство секции,
	- на секции должен быть оверлей поверх основного фона
	- контент секции всегда центрируется
	- <768px становится в одну колонку и центрируется по центру блока 

3) Секция "Yoga FEATURE PROGRAM"
	- контент секции центрируется
	- текст и картинка центрируются относительно друг друга 
	- у блоков высота не фиксированная
	- соседние блоки должны быть одинаковой высоты при добавлении текста 
	- <768px влоки переворачиваются вертикально 
	- <570px блоки должны стать в один столбик
	- кликабельные элементы 

4) Секция с картинками, которые заполняют все пространство страници
	- картинки должны быть кликабельные и при ховере должна плавно появляться иконка с оверлеем 
	- картинки должны пропорционально ресайзиться при изменении окна браузера
	- <768px картинки тановятся в 2 колонки 

5) Секция "About us"
	- имеется 2 колонки, слева контент с текстом, справа слайдер slick
	- слайдер подключаем так:
		https://kenwheeler.github.io/slick/
		https://bezramok-tlt.ru/blog/posts/slick-slider-ustanovka-nastrojka-primery-i-dokumentaciya/
	- стрелки в слайдере центрируются по высоте при любом размере блока
	- <768px все перестраивается в один столбец 
	- стрелки не должны обрезаться

6) Секция info
	http://joxi.ru/Drl0RR0C095aZr
	- фон должен растягиваться на всю секцию, оверлей должен тянуться на всю секцию
	- блочки в одном ряду должны быть одинаковой высоты и располагаться по 4 в ряду 
	- при уменьшении или при увеличении колличества блочков, они должны центирроваться на странице
	- при уменьшении окна браузера , текст блочков не должен слипаться и не должен прилипать к краю окна браузера
	- <768px надо перестроить в 2 колонки 
	- <570px одна колонка http://joxi.ru/DmBPqqPiz0vdzA

7) Секция с карточками
	фон блока спозиционирован в правый нижний угол
	карточки должны располагаться по 3 в ряду
	высота карточек не фиксируется
	< 992px перестраиваются карточки по 2 в ряд
	< 768px перестраиваются карточки по 1 в ряд 

8) footer
	для адресса используем правильный тег address
	для телефонов и тдю тоже используем правельные теги
	иконки вставляем через icomoon https://icomoon.io/
